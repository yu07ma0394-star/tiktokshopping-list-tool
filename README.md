# 発送リスト作成ツール

TikTok Shop の注文エクスポート（Excel / CSV）を読み込んで、宛名・住所・商品・個数の**発送リスト**を作り、そのままA4で印刷できる単体HTMLツールです。

## 特長
- インストール不要。ブラウザで開くだけで動作
- データは端末内（ブラウザ内）だけで処理され、外部に送信されません
- 複数ファイルの結合、商品別の集計、印刷 / PDF保存に対応
- 列の順番が違っても見出し名で自動判別

## 使い方
1. 公開ページ（GitHub Pages）を開く、または `index.html` をブラウザで開く
2. Excel / CSV ファイルをドラッグ＆ドロップ（複数・追加OK）
3. 「リストを作成」→「印刷 / PDF保存」

## オプション
- 電話番号の表示ON/OFF
- 〒の区切り表示（812-0053 形式）
- 商品名フル表示
- 並び順（商品順 / 宛名順 / ファイル順）

## 商品の短縮名の追加
`index.html` 内の `SKU_SHORT` に、セラーSKUと表示名を追記できます。

```js
const SKU_SHORT = {
  'wo01':'ウィッグオイル 300mL',
  'ri_se01':'姿勢サポーター',
  // 追加例:
  // 'あなたのSKU':'表示したい商品名',
};
```
